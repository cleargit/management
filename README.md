# management

#### 项目介绍
Springboot + shiro权限管理 博客后台管理。

#### 开发环境

| 工具    | 版本或描述                |    
| ----- | -------------------- |    
| OS    | Windows 10            |    
| JDK   | 1.8                 |    
| IDE   | IntelliJ IDEA 2018.1 |    
| mysql | 5.7                |    
| redis |  3.2            |
